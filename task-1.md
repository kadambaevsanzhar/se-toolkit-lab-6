# Task 1 — Call an LLM from Code

## Goal
Implement a Python script that calls an LLM via API.

## Steps
1. Load environment variables from `.env.agent.secret`
2. Configure LLM API endpoint
3. Send a prompt to the LLM
4. Receive and print the response
5. Add a regression test

## Expected Result
Running the script should call the LLM and print the response.
